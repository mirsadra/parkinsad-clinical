How should the BPMH panel work?

Three approaches — different entry points into the workflow

A

Full Workflow Replacement

Tech enters community meds from SystmOne/HIE into the app before going bedside. App compares against inpatient FHIR meds, highlights discrepancies, and produces a single combined printout. After bedside, tech marks confirmations and stock status. App generates the Task note text ready to paste into PowerChart.

Pros

  • Replaces two printouts with one structured sheet
  • Discrepancies caught before bedside
  • Task note fully auto-generated
  • Data captured once, used everywhere

Cons

  • Requires data entry before going bedside
  • Biggest behaviour change for technicians
B

Post-Bedside Task Note Generator

Don't touch the pre-work. Tech still prints from SystmOne/HIE as today. After bedside, they open the app and fill a structured form: sources consulted, discrepancies found, stock adequacy, pharmacy, OTC/alcohol/tobacco. App generates the PowerChart Task note text and a BPMH entry order list.

Pros

  • Minimal change to existing workflow
  • Immediate value — eliminates note-writing time
  • Easy to adopt

Cons

  • Discrepancy spotting still manual
  • Doesn't fix the pre-work problem
C

Pre-work Comparison Tool (Start Here)

Focus only on solving A first. Tech enters community meds into app (from SystmOne/HIE on another screen). App auto-compares with inpatient FHIR meds and highlights discrepancies. Produces a single annotatable printout. Task note generation comes in a future step.

Pros

  • Solves the highest-risk problem (discrepancy spotting) first
  • Smaller, shippable piece
  • Natural foundation for A

Cons

  • Task note still manual
  • Doesn't feel complete on its own

My recommendation is A — it's more work upfront but the pre-work entry feeds everything else. The discrepancy comparison and Task note generation are both impossible without that data in the app. B gives quick wins but leaves the hardest problem (A) unsolved. C is the right first chunk of A if A feels too big to ship at once.
